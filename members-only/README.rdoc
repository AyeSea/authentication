== README

Repository for "Project: Authentication" from The Odin Project

Members Only Clubhouse:
Create a microblog using Rails where members can write short posts about non-members. Only logged in members can verify the authors of the posts, while the rest of the public is left guessing.

Visit http://www.theodinproject.com/ruby-on-rails/authentication?ref=lc-pb for more information on this particular project.
